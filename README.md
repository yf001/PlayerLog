# PlayerLog
プレーヤー関連のログを取ります。<br/>

現在はブロックの設置、破壊、プレーヤーの実行したコマンドなどのログが取れます。

今後はブロックタッチのログや設置された看板のログなども対応予定

###現在使用可能なログのファイル形式
sqlite(.sqlite3)のみ<br/>
